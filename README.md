# PLC OPC UA 通讯与 HMI 组态设计器

一个基于 **C# + WPF + MVVM + OPC UA** 的工业上位机 / HMI 组态平台原型。

当前版本已经具备：

- OPC UA 通讯
- 变量导入与自定义变量管理
- 事件绑定
- 拖拽式组态设计器
- 多页面项目
- 运行态 / 设计态强化隔离
- 工业模块图形化表达
- 更正式的运行页与手动画面
- 参数设定行级权限只读
- 报警确认 / 复位 / 恢复 / 历史持久化
- 参数持久化

---

## 一、核心功能

### 1. OPC UA 通讯
- IP / 端口 / Endpoint 可配置
- 匿名 / 用户名密码连接
- 支持读写变量
- 支持订阅优先 / 轮询兜底

### 2. 变量系统
- CSV 变量表导入
- 自定义变量新增
- 变量分类、分组、方向、报警、可写属性

### 3. 事件绑定
- 变量与事件关联
- 支持触发条件、事件名、目标、参数
- 报警类事件支持触发、恢复、计数

### 4. 组态设计器
- 左侧工具箱
- 中间画布
- 右侧属性编辑器
- 支持拖拽添加控件
- 支持控件位置调整、复制、粘贴、删除
- 支持页面模板应用
- 运行态下禁止组态编辑，进一步隔离设计器与运行页

### 5. 运行态页面
- 正式运行总览
- 手动画面
- 参数设定
- 报警画面
- 登录权限

### 6. 工业模块
当前支持：
- Motor
- Cylinder
- Axis
- Robot
- Stopper
- Indicator
- ValueDisplay
- AlarmBanner
- PageButton

### 7. 手动画面与设备控制
支持：
- 电机启停
- 电机故障复位占位输出
- 气缸伸缩
- 气缸前到位 / 后到位状态显示
- 挡停升降
- 机械手启停
- 机械手暂停占位
- 机械手复位占位
- 轴使能
- 轴报警复位
- 轴 Jog+
- 轴 Jog-
- 轴定位移动
- 轴回零

### 8. 参数与报警
- 参数分页展示
- 参数行级权限只读控制
- 参数持久化到 `config/parameters.json`
- 报警历史持久化到 `config/alarm-history.json`
- 报警确认
- 报警复位
- 报警恢复记录
- 报警次数统计
- 未确认报警计数

### 9. 权限控制
角色：
- 操作员
- 工程师
- 管理员

示例密码：
- 工程师：`123456`
- 管理员：`admin888`

## 四、配置文件

默认保存在项目目录下：

- `config/appsettings.json` 通讯与变量配置

- `config/designer-project.json` 组态工程
- `config/parameters.json` 参数文件
- `config/alarm-history.json` 报警历史
